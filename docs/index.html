MemoryAI Coding-Guard

Universal coding-agent guardrails. Install once → 11 IDE/CLI follow the rules.

Install

npx @memoryai.dev/coding-guard install

Detects which AI agent your project uses (Cursor, Claude Code, Aider, Copilot, …) and drops the right rule files into the right paths with the right format.

The 4 principles

Think Before Coding

State assumptions. Surface tradeoffs. Don't pick silently.

Simplicity First

Minimum code. No speculative abstractions.

Surgical Changes

Touch only what you must. Match existing style.

Goal-Driven

Define success criteria. Loop until verified.

Plus 5 anti-sloppy rules

Anti-hallucinate · Anti-fake-verify · Anti-out-of-scope · Anti-loop · Anti-destructive
